# CLI Task Tracker
A simple Command Line Interface (CLI) built with Python to manage your daily tasks.

## Features
- **Add** new tasks with a description.
- **List** all tasks or filter by status (`todo`, `in-progress`, `done`).
- **Update** task status or description.
- **Delete** tasks by ID.
- **Persistence**: All data is saved in a local `tasks.json` file.

## How to Run
1. Clone the repository:
   `git clone https://github.com/your-username/task-tracker-cli.git`
2. Run the script:
   `python task_tracker.py add "New Task"`
